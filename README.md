# Candidate Intelligence Engine

Workspace for the Redrob Intelligent Candidate Discovery & Ranking Challenge.

## Layout

| Path | Purpose |
|---|---|
| `data/` | Candidate pool, sample candidates, and JSON schema |
| `docs/source/` | Original challenge documents from the bundle |
| `docs/analysis/` | Analysis deliverables and feature-engineering notes |
| `submissions/` | Submission examples and generated ranking CSVs |

## Key Files

- `data/candidates.jsonl`: full candidate pool.
- `data/sample_candidates.json`: small formatted candidate sample.
- `data/candidate_schema.json`: schema for candidate records.
- `docs/source/job_description.docx`: target role description.
- `docs/source/redrob_signals_doc.docx`: behavioral signal reference.
- `docs/source/submission_spec.docx`: submission format and evaluation rules.
- `docs/analysis/feature_hypothesis.md`: Phase 1 feature-engineering blueprint.
- `submissions/sample_submission.csv`: format-only sample submission.
